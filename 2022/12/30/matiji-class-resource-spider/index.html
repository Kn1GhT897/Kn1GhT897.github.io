码蹄集-课程资源爬虫

发表于2022-12-30|更新于2022-12-30
|阅读量:

为什么写它

作为百度松果菁英班（虽然也不知道有啥用就是之前假期无聊报名的个劳什子）的一员，我们不难发现班级详情中资源一栏中有不少不错的课程视频，但是又不想打开浏览器看这些视频的我，只好把他们全部下载下来看。本着能自动就不动手的原则，我写了这个爬虫。

详情

初见网页

点进资源这一栏，大概的模式如下图所示。点击查看按钮之后，会出现一个播放器，同时 URL 未改变，我们容易知道这是一个动态网页。

开始分析

点开视频，同时打开 dev tools，我们可以很容易从网页源代码中的找到这个视频对应的 URL，其形式为:

https>//cdn.matiji.net/YYYY-MM-DD/video-name.mp4?auth_key=balabala

有了这个 URL，大概的思路也就有了：抓出来这个 URL 是怎么生成的。

在 dev tools 的帮助下，我们直接搜索这个 URL，经过筛选，可以发现是通过 https://matiji.net/exam-back/api/downUploadFileById.do 这个接口直接获得的 URL。

查看它的 payload，发现含有两个元素: id 和 classid。classid 应该就是我们所在课程的号码，但是 id 暂时不知道是什么。

继续在 dev tools 中筛选，我们发现通过 https://matiji.net/exam-back/api/queryClassFileList.do 接口，获取了所在课程的资源，查看 payload 得知有三个元素 start,limit 和 classid。那么通过这些信息，我们可以构建脚本了。

代码环节

我的环境是 Ubuntu22.04LT + Python3.10.6，其中 Python 依赖只有 httpx，Ubuntu 有 wget 这个 cli 工具。
